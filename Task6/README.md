# Signup Form Project

# Project Description

This project is a responsive and user-friendly Signup / Registration Form developed using HTML, CSS, and JavaScript.
It allows users to create an account by entering their personal details with proper client-side validation and real-time feedback.
The main goal of this project is to demonstrate form validation, modern UI design, and interactive user experience using front-end technologies.
# Features
Clean and modern user interface
Two-panel layout with welcome section and form section
Client-side form validation
Real-time password strength indicator
Show / Hide password functionality
Floating labels for input fields
Error messages for invalid inputs
Success message on valid submission
Fully responsive design (desktop & mobile friendly)
# Technologies Used
HTML – for page structure
CSS – for styling, layout, and responsiveness
JavaScript – for validation and interactive functionality
Form Fields
The signup form collects the following user information:
First Name
Last Name
Phone Number
Email Address
Password
Confirm Password
User Type (Student / Admin)
# Validation Rules
All fields are mandatory
Email must be in valid format
Phone number must be exactly 10 digits
Password must:
Contain at least 8 characters
Include one uppercase letter
Include one special character
Confirm password must match the password
User type must be selected
Password Functionality
Password strength is shown as Weak, Medium, or Strong
Users can toggle password visibility
Ensures better security and usability
User Feedback
Error messages are displayed below respective fields
Success message is shown after successful validation
Form submission is prevented if validation fails
# Responsive Design
Layout automatically adjusts for smaller screens
Mobile-friendly design using media queries
Project Use Case
This project can be used as:
College mini-project
Front-end practice project
Signup page template for websites
Learning example for form validation
